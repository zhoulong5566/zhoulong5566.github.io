01_数据库介绍和命令行操作

Posted on 2019-10-26 | In MySQL

Java主要是对计算机内存中的数据进行操作的。
MySQL一方面便于对数据进行MySQL可以永久的将数据保存在数据库系统中。
另一方面便于对数据进行增删改查操作。
本文主要是关于数据库命令行操作的介绍

MySQL简介：

数据库作用：数据持久化 + 便于查询
相关名词：
    DB:database
    DBMS:database management system
常见数据库管理系统：
    MySQL、Oracle、DB2、SQL server等
数据库管理系统分类：
    基于共享文件系统的DBMS：Access
    基于客户机——服务器的DBMS：MySQL、Oracle、SQL server
SQL：Structure Query Language
SQL语言分类：
    DML(Data Manipulation Language):数据操纵语句
        insert、update、delete、select
    DDL(Data Definition Language)：数据定义语句
        create table、alter table、drop table、create index、drop index
    DCL(Data Control Language):数据控制语句
        grant、revoke、commit、rollback、savepoint、lock

MySQL命令行操作：

启动和停止MySQL服务：
    方式一：
        右键计算机 > 管理 > 服务 > 启动或停止MySQL服务
    方式二：
        启动：net start mysql 服务名（服务名可以通过服务来查看）
        停止：net stop mysql 服务名
MySQL服务端的登陆和退出
    登陆：mysql -h 主机名 -P 端口号 -u 用户名 -p 密码
    退出：exit

--------------------------------------------------------------
进入MySQL：
    mysql -uroot -p#### (#表示密码)
使用一个数据库：
    use 数据库名称;
新建一个数据库：
    create database 数据库名;
进入数据库后查看有哪些数据表：
    show tables;
建表：
    create table stu(
        id varchar(30),
        age int,
        name varchar(30),
        birthday date
    );
查看表结构：
    desc 表名;
删除表：
    drop table 表名;
--------------------------------------------------------------
查看表中所有记录：
    select * from 表名;
向表中插入数据：
    insert into 表名(列名列表) values(列对应的值的列表);
    注意：插入varchar或date型数据要用 单引号 引起来
修改表中记录：
    update 表名 set 列1=列1的值，列2=列2的值 where...
删除表中记录：
    delete from 表名 where ...
    delete from stu where id=1;
查询所有的列：
    select * from 表名;
查询特定的列：
    select 列名1,列名2, ... from 表名;
对查询的数据进行过滤：
    使用where字句
运算符：
    select * from customer
    where salary >= 2000 and salary <= 3000;

    select * from customer
    where salary between 2000 and 3000;

    select * from customer
    where salary = 2000 or salary = 3100;

    select * from customer
    where salary in (1000,2000,3100);

    -----------------------------------
    % 匹配任意多个字符；
    _ 只匹配一个字符

    # 查询name中有o的人的名字
    select * from customer
    where name like '%o%'; 

    # 查询name中第3个字母是r的人的名字
    select * from customer
    where name like '__r%';

    # 查询email为空的所有人的信息
    select * from customer
    where email is null;

    # 查询email不为空的所有人的信息
    select * from customer
    where email is not null;

    # 查询所有客户的信息，且按salary升序排列(默认升序)
    select * from customer 
    order by salary asc;

    # 查询所有客户的信息，且按salary降序排列
    select * from customer 
    order by salary desc;

MySQL数据查询：

查询一：基础查询
    语法：
        select 查询列表 from 表名 where...；
    说明：
        select： 选择查询哪些列（字段）
        from: 从哪查询
        where： 表示过滤
    特点：
        查询列表可以是：表中字段、常量值、表达式、函数
        查询结果是一个虚拟的表格
